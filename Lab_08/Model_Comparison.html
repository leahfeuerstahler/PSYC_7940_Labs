Overview

In previous labs, we learned how to use brms to set priors, fit models, and summarize results.

Log Likelihood

We can use log_lik to get the log density of the model for each iteration and each person.

  • This is a measure of how well the data fit to the model.
res_l <- log_lik(mod1)

The output is pointwise, meaning that each of the 4000 MCMC iterations and each of the 179 individuals in the data has its own contribution to the log density:

res_l %>% glimpse

Out-of-Sample Deviance

The log pointwise density is a measure of goodness-of-fit in the current sample.

We usually care more about how the model performs in new samples.

We will learn about two methods to compare the relative performance of models in new samples:

  1. loo: Approximate leave-one-out cross-validation via Pareto-smoothed importance sampling (PSIS)

  2. waic: Balances in-sample prediction with a penalty term, similar to other information criteria

Out-of-Sample Deviance

  • Pareto-smoothed importance sampling (PSIS) is computationally efficient approximation to leave-one-out cross-validation.

  • Exact leave-one-out cross-validation would involve fitting the model \(N\) times, once for each of the \(N\) rows in the data!

  • loo is pointwise. We get one set of fit information for each row in our data:

loo_res <- loo(mod1)

Model Comparison

Back to the cycling data. We will fit new models that incorporate information about riders’ specialties.

  • Leader: riders who aim to win the entire 3-week Tour-de-France

  • Climber: riders who focus on results for hilly and mountainous terrain

  • Sprinter: riders who focus on results for flat terrain

  • All-rounder: riders who have no particular specialty

table(TDF$Specialty)

All-rounder     Climber      Leader    Sprinter 
         91          32          23          33

Model Comparison

Let’s add Specialty as an index variable intercept into our model.

brms_form <- Weight ~ 0 + Specialty + HeightC

brms_prior<-prior(normal(66.5, 2), class = "b") +
            prior(normal(1, .5), class = "b", coef = "HeightC") +
            prior(exponential(.25), class = "sigma")

mod2 <- brm(brms_form,
            data = TDF,
            family = gaussian(link = "identity"),
